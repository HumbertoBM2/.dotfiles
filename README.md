## Windows

### Neovim

Nvim config folder needs to be located at: 

```bash
$env:LOCALAPPDATA\nvim
```

### PowerShell

Open a terminal and type the following command:

```bash
notepad $PROFILE
```
In the notepad, paste the contents of `Microsoft.PowerShell_profile.ps1`
